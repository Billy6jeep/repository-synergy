# DNShell v1.7

#### Description:
  A Python Reverse Shell that uses DNS as the C2 channel.


#### Requires:
  - dnslib
  - dnspython
  - pycrypto

#### Gotchyas:
  - don't forget to change your HOST
  - don't forget to change your SECRET key
  - run server first
