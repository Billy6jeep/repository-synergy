Xiami For Linux Project
=======================
模仿虾米音乐 Mac 版本

在使用 Ubuntu 过程中，不满没有喜欢的音乐播放器，一直使用虾米音乐于是写下

本项目原意只是 PyQt4 学习实验性的项目，现在也有继续写的欲望
